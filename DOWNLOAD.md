* GNS3 (latest version recommended)

## 🚀 How to Run the Lab

1. Download the project archive
2. Import the `.gns3` file into GNS3
3. Ensure required images are configured
4. Start all devices
5. Run verification commands (see README)

---

## 📌 Notes

* Ensure all interfaces are enabled (`no shutdown`)
* Verify OSPF neighbors are in FULL state
* Check MPLS LDP and BGP sessions before testing
